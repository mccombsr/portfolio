iClean

Basic Web Application to allow small/ independent cleaning businesses to post their information where potential clients can search for house cleaners by zip code.

  • Built using ReactJS and NodeJS
  • Managed state using Redux
  • Handled user login with Auth0
  • Used PostgreSQL for Database
  • Styled using SASS, FlexBox and CSS Grid
  • Set up Amazon S3 and Dropzone for storing business logos

TeachTools

Web Application geared towards assisting teachers in providing a fun way to review materials taught. Inspired by Jeopardy, the application allows teachers to create trivia games with categories, set up teams and assign points for questions answered correctly.

  • Built using ReactJS and NodeJS
  • State shared across components with Redux
  • Authentication handled with B-crypt
  • Socket.io used to set up rooms by game
  • Database managed with PostgreSQL
  • Styled using SASS, FlexBox, CSS Grid, and Ant-Design
